# `@ezweb/ts-helpers`
[![npm version](https://badge.fury.io/js/@ezweb%2Fts-helpers.svg)](https://badge.fury.io/js/@ezweb%2Fts-helpers)


Add this to your `tsconfig.json` file
```json
{
  "ts-node": {
    "require": [
      "@ezweb/ts-helpers"
    ]
  },
  "compilerOptions": {
    "types": [
      "@ezweb/ts-helpers"
    ]
  }
}
```
