### Install
```shell
nvm install
nvm use
npm i
```

### Tests
```shell
npm run compile # Must compile before testing
npm run test
```

## Bump version
```shell
npm version patch -ws
npm version prerelease -ws
```

### Publish
```shell
# all
npm run prepublish
npm publish -ws

# publish just one ?
npm publish -w @ezweb/ts-helpers
```
